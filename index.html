Snowflake Catcher

Left click to draw red stuff that catchs the snowflakes. Right click to erase the red stuff. Press space bar to redraw the screen. There are two ways of producing the same result. Every time the screen is redrawn, one of two modes is selected randomly, which runs the program using two different methods. the slower one uses the get() function while the faster one uses the more efficient pixels[] array.
